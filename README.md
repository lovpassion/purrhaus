## ✨ What is Purrhaus?

Purrhaus is a small desktop app that turns your computer into a private, good-looking vault for your
files and notes. Everything stays **on your machine** — there's no sign-up, no server, and nothing is
uploaded unless *you* explicitly choose to share something.

It looks and feels like a modern file explorer, with a kawaii cat theme, a built-in Markdown note
editor, encryption, and a bunch of quality-of-life touches. The interface speaks **English or French**
and picks your system language automatically on first launch (changeable anytime in Settings).

## 📥 Get started

1. **Download `Purrhaus.exe`** from this repository (click the file above, then **Download**).
2. **Double-click it.** A real "Purrhaus" window opens — no browser, no terminal.
3. A cat icon appears in your Windows tray. Closing the window tucks it into the tray; right-click the
   icon for **Open / Restart / Start with Windows / Quit**.

> **Requirements:** Windows 10 or 11 with the **WebView2 runtime** (already installed on up-to-date
> systems). The app is self-contained — nothing else to install.
>
> Windows may show a "Windows protected your PC" notice for a new unsigned app. Click
> **More info → Run anyway** if you trust the download.

## 🎀 Features at a glance

**Files & folders**
- Drag & drop import (files *and* whole folders), nested folders, breadcrumbs.
- Multi-select (lasso, Ctrl/Shift+click), move, rename, and undo.
- Download a single file or a whole selection as a `.zip` (folder tree preserved).
- Tags, favorites, smart folders (saved searches), trash with restore & auto-purge.
- Duplicate finder and batch rename.

**Notes & editor (Obsidian-style)**
- Markdown editor with `[[wiki-links]]`, backlinks, outline, and a notes **graph**.
- Daily notes, templates, task lists, live split preview, and image/note embeds.
- Syntax highlighting, with an optional advanced editor (line numbers, code folding, multi-cursor).

**Media**
- Inline previews for images, audio, video, PDF and text/code.
- Built-in image editor (rotate, crop, flip, resize), fullscreen slideshow, a floating audio player,
  and webcam/microphone capture.

**Privacy & security**
- **Encryption at rest** (AES-GCM) protected by a passphrase that's never stored.
- **App lock** with auto-lock, EXIF metadata cleaning, and secure delete.
- Strictly local: no telemetry, no third-party scripts.

**Nice touches**
- 8 themes (light + dark) plus an Auto mode that follows your system, and a custom theme editor.
- Command palette (`Ctrl/Cmd + K`), dashboard, calendar, achievements, and a friendly mascot.
- Backup & restore your whole vault to a (optionally encrypted) archive.

**Sharing (only when you ask for it)**
- Temporary share link with a QR code, device-to-device transfer over your own Wi-Fi, or an encrypted
  standalone file.

## ❓ FAQ

**Where are my files stored?**
> Locally, inside the app's private storage on your computer. Nothing is uploaded automatically.

**Is anything sent to the internet?**
> Only if you deliberately use a sharing feature. Day-to-day use is fully offline.

**I set an encryption passphrase and forgot it. Can it be recovered?**
> No — by design. The passphrase is never saved, so keep it somewhere safe.

**Can I move my vault to another computer?**
> Yes. Use **Settings → Backup & restore** to export an archive, then restore it on the other machine.

**Does it work on macOS or Linux?**
> This download is for Windows. (The app is web-based under the hood, so other platforms may come later.)
